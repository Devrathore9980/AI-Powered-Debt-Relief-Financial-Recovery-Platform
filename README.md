# 💰 AI-Powered Debt Relief & Financial Recovery Platform

An AI-powered web platform that helps individuals analyze their debt, understand their financial situation, receive personalized repayment strategies, and generate AI-assisted settlement recommendations. The platform aims to simplify debt management and support users in achieving financial stability.

---

## 🚀 Project Overview

Managing debt can be stressful and confusing. This platform uses Artificial Intelligence and financial analysis to provide users with personalized debt recovery solutions.

The system analyzes:
- Loan information
- Financial profile
- Income & expenses
- Repayment capacity
- Debt-to-income ratio
- Settlement possibilities

Based on this data, AI generates customized recommendations to help users reduce financial burden and recover faster.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/AI_Powered_Debt_Relief_Financial_Recovery_Platform.git
```

```bash
cd AI_Powered_Debt_Relief_Financial_Recovery_Platform
```

---

## Frontend Setup

```bash
cd frontend
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

---

## Backend Setup

```bash
cd backend
```

Create virtual environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run server

```bash
uvicorn main:app --reload
```
